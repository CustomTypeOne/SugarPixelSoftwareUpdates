# SugarPixel Software Updates

This repository is a historical archive of SugarPixel software update versions. 

Follow the SugarPixel User Guide to update your software. 
https://www.CustomTypeone.com/SugarPixelHelp

For most people, updating just needs to follow the below steps:
1. Download the latest version
2. Unzip the downloaded file
3. Copy the update.bin file to a micro sd card
4. Insert the sd card into the back of SugarPixel
5. Unplug SugarPixel and them plug it in again to restart
6. Wait for the screen to show Update Success, Reboot
7. Restart SugarPixel again
